# NotesProject
# 📝 Personal Notes App

A clean, responsive, and fully functional notes application built with pure HTML, CSS, and Vanilla JavaScript. Designed with a focus on simplicity, featuring a modern pastel UI and a seamless dark mode toggle.

## ✨ Features(till now)

* **Complete CRUD Functionality:** Create, read, pin, and delete notes instantly.
* **Real-time Search:** Filter notes by title or content dynamically as you type.
* **Dark/Light Theme:** Smooth transition between a light pastel aesthetic and a sleek dark mode.
* **Responsive Grid Layout:** Automatically adapts to mobile, tablet, and desktop screens using CSS Grid.
* **No Dependencies:** Built entirely with vanilla web technologies (no heavy frameworks).

## 🛠️ Tech Stack(till now)

* **HTML5:** Semantic structure and layout.
* **CSS3:** Custom properties, Flexbox, CSS Grid, and responsive design without media queries.
* **Vanilla JavaScript:** DOM manipulation, state management, and event handling.
* **FontAwesome:** Scalable vector icons.

## 🚀 How to Run Locally

1. Clone the repository to your local machine.
2. Open the project folder.
3. Simply double-click the `index.html` file to open it in your default web browser. No server or build process required!
